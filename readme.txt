About the Website:-

BELLY UP is a restaurant reservation website created for Assignment 2 in COS10005 Web Development. The website allows users to browse restaurants, make reservations, register as users, and recommend new restaurants.

Pages Included:-

Home (index.html)
- Introduction to the website and navigation to all pages.

Restaurants (restaurants.html)
- Displays restaurant information.
- Users can click "Reserve Table" to make a booking.

Reservation (reservation.html)
- Reservation form for booking a table.
- The selected restaurant is automatically filled in when coming from the Restaurants page.
- Deposit amount changes depending on the restaurant selected.

Register (register.html)
- Registration form with validation.

Recommend (recommend.html)
- Allows users to recommend a restaurant.

About (about.html)
- Information about the project and developer.

CSS Techniques Used:-

The website styling was created using CSS. Some of the techniques used include:

- Flexbox for navigation bars and page layouts.
- CSS Grid for restaurant card layouts.
- Hover effects on buttons, cards and restaurant images.
- Responsive design using media queries.
- Border radius and box shadows for modern card designs.
- Custom colours, fonts and spacing for visual consistency.
- Background effects and transitions for improved user experience.

JavaScript Features Used:-

JavaScript was used to improve interactivity and form functionality. Features include:

- Form validation for registration and reservation forms.
- Checking required fields before submission.
- Automatic restaurant selection from the Restaurants page using URL parameters.
- Dynamic deposit calculation based on selected restaurant.
- Event handling using onchange and onsubmit.
- Displaying and hiding form elements when needed.
- Copying email information when selected by the user.
- Basic DOM manipulation using getElementById() and query selectors.

Testing:-

The website was tested using Google Chrome and Safari. All links, forms, navigation menus and JavaScript functions were checked to ensure they work correctly.

Challenges Faced:-

Some challenges included linking pages correctly, ensuring CSS loaded properly on the Mercury server, and transferring restaurant information between pages. These issues were resolved through testing and debugging.
